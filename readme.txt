============================ CADASTRO DE FUNCIONÁRIOS ============================

Esse é um projeto simples em Python para cadastro de funcionários que visa reiterar
 o uso de estruturas de repetição em um projeto visualmente simples fazendo transição
de tela em um primeiro processoe uma função de espera (.sleep(2)) de 2 segundos 
para, então, transicionar para uma última tela que apresentará todos os dados do 
Cadastro de Colaborador realizado.

O programa também visa trabalhar com responsividade tanto em sistemas Windows
quanto Linux, fazendo a adição de operador ternário, para a limpeza de tela que
proporciona o efeito de transição.
