============================ CADASTRO DE FUNCIONÁRIOS ============================

Esse é um projeto simples em Python para cadastro de funcionários que visa reiterar
 o uso de estruturas de repetição em um projeto visualmente simples fazendo transição
de tela em um primeiro processo e uma função de espera (.sleep(2)) de 2 segundos 
para, então, transicionar para uma última tela que apresentará todos os dados do 
Cadastro de Colaborador realizado.

O projeto ainda vislumbra a verificação de idade máxima não aceitando valores maiores
que 150.

O programa também é pensado em conforto visual ao transicionar da tela de inserção de 
dados para o resultado dos dados inseridos no cadastro, respondendo nos principais 
sistemas operacionais, fazendo a adição de operador ternário para a limpeza/transição 
de tela, proporcionando uma visualização melhor dos dados inseridos.
